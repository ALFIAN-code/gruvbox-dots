## Instalation

Clone the project
```bash
  git clone https://github.com/ALFIAN-code/hyprland-gruvbox-dots.git
```

open folder
```bash
cd hyprland-gruvbox-dots
```

run script instalation
```bash
  ./setup.sh
```
